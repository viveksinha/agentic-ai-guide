What It Is

Microsoft Copilot Studio is the no-code platform inside Microsoft 365 for building AI agents. You describe what you want your agent to do in plain English, and it builds the setup. You then deploy it to Teams, Outlook, or a website with a single click.

It's already included in your Microsoft 365 subscription — no extra purchase needed.

What Makes It Special

  • ✓ No code — describe agent in plain English
  • ✓ One-click deploy to Microsoft Teams
  • ✓ Native Outlook Calendar & Email connectors
  • ✓ Connects to SharePoint knowledge bases
  • ✓ Pre-built insurance agent templates
  • ✓ Can use Claude as the AI model

How to Get Started — Step by Step

  1. 1

    Open Copilot Studio

    Go to copilotstudio.microsoft.com and sign in with your Microsoft 365 work account. It's already included — no sign-up needed.

  2. 2

    Create a New Agent

    Click "Create" → "New Agent". Describe your agent in plain English, for example:
    "You are an annuity advisor assistant. Ask customers about their age, retirement timeline, and investment amount, then recommend the right product."

  3. 3

    Choose Your AI Model (Optional)

    Go to Agent Settings → Model. You can select Claude Sonnet instead of GPT if you prefer Anthropic's responses.

  4. 4

    Add Actions (Connect to Real Systems)

    Go to "Actions" → "Add an action". Search for "Office 365 Outlook" to add calendar booking and email sending — all through a visual UI, no code.

  5. 5

    Deploy to Microsoft Teams

    Go to "Channels" → "Microsoft Teams" → "Turn on Teams" → "Publish". Your agent appears in Teams within minutes. Pin it to the sidebar so everyone sees it.

What It Is

Claude is Anthropic's AI assistant — available at claude.ai. For building agents, you use Claude's Projects feature: create a project, write your agent's instructions in plain English, and share the project link with your team.

For a fully custom application (web widget, CRM integration), a developer uses the Claude API to build it — about 50 lines of Python for a working agent.

What Makes It Special

  • ✓ Agent up and running in 5 minutes
  • ✓ No setup, no installation, no API key needed
  • ✓ Projects = persistent agent memory + instructions
  • ✓ Share a project link with your whole team
  • ✓ Best-in-class reasoning and instruction-following
  • ✓ Works as the AI brain inside Copilot Studio
ℹ️

Two ways to use Claude: (A) claude.ai — no code, works right now, great for prototyping and small teams. (B) Claude API — for developers building a proper app (web widget, CRM integration). See the How-To Guide for full code examples.

Option A — No Code: Claude Projects (5 Minutes)

  1. 1

    Go to claude.ai

    Open claude.ai and sign in. Free accounts work fine for prototyping.

  2. 2

    Create a New Project

    Click "Projects" in the sidebar → "New Project". Give it a name like "Annuity Advisor Agent".

  3. 3

    Add Instructions

    Click "Project Instructions" and paste your agent's instructions in plain English. See the How-To Guide for the full example prompt.

  4. 4

    Test and Share

    Start a new chat inside the project — Claude now behaves as your agent. Click "Share" to give teammates access via a link.

Option B — Developer: Claude API

bash — Install 
pip install anthropic
python — Minimal Agent 
import anthropic

client = anthropic.Anthropic(api_key="your-api-key")

SYSTEM_PROMPT = """
You are an insurance assistant.
[Describe your agent's job here in plain English]
"""

history = []

while True:
    user_input = input("You: ")
    history.append({"role": "user", "content": user_input})

    response = client.messages.create(
        model="claude-opus-4-5",
        max_tokens=500,
        system=SYSTEM_PROMPT,
        messages=history
    )

    reply = response.content[0].text
    history.append({"role": "assistant", "content": reply})
    print(f"\nAgent: {reply}\n")
